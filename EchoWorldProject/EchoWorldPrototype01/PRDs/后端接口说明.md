# EchoWorld 后端接口说明

## 概述
本文档描述了 EchoWorld 应用的后端 API 接口规范。后端服务地址：`https://showx-api-server-820411680197.us-west1.run.app`

## 接口分类

### 1. 认证相关接口 (Auth)

#### 1.1 用户注册
- **接口地址**: `/api/auth/register`
- **请求方法**: `POST`
- **请求参数**:
  ```json
  {
    "email": "user@example.com",
    "password": "password123"
  }
  ```
- **响应数据**:
  ```json
  {
    "player": {
      "id": "player-uuid",
      "email": "user@example.com",
      "name": "",
      "avatarUrl": null,
      "gender": "undisclosed",
      "createdAt": "2024-01-01T00:00:00.000Z"
    },
    "tokens": {
      "accessToken": "jwt-token",
      "refreshToken": "refresh-token",
      "expiresAt": 1704067200000
    }
  }
  ```

#### 1.2 用户登录
- **接口地址**: `/api/auth/login`
- **请求方法**: `POST`
- **请求参数**:
  ```json
  {
    "email": "user@example.com",
    "password": "password123"
  }
  ```
- **响应数据**:
  ```json
  {
    "player": {
      "id": "player-uuid",
      "email": "user@example.com",
      "name": "Player Name",
      "avatarUrl": "https://example.com/avatar.jpg",
      "gender": "male",
      "createdAt": "2024-01-01T00:00:00.000Z"
    },
    "tokens": {
      "accessToken": "jwt-token",
      "refreshToken": "refresh-token",
      "expiresAt": 1704067200000
    }
  }
  ```

#### 1.3 邮箱验证
- **接口地址**: `/api/auth/verify-email`
- **请求方法**: `POST`
- **请求参数**:
  ```json
  {
    "token": "verification-token"
  }
  ```
- **响应数据**:
  ```json
  {
    "success": true
  }
  ```

#### 1.4 检查邮箱是否存在
- **接口地址**: `/api/auth/check-email`
- **请求方法**: `POST`
- **请求参数**:
  ```json
  {
    "email": "user@example.com"
  }
  ```
- **响应数据**:
  ```json
  {
    "exists": false
  }
  ```

### 4. 消息相关接口 (Messages)

#### 4.1 获取消息列表
- **接口地址**: `/api/characters/{characterId}/messages`
- **请求方法**: `GET`
- **请求头**: `Authorization: Bearer {accessToken}`

## 错误处理
所有接口可能返回以下错误格式：

```json
{
  "error": {
    "code": "INVALID_REQUEST",
    "message": "请求参数无效"
  }
}
```

常见错误码：
- `INVALID_REQUEST`: 请求参数无效
- `UNAUTHORIZED`: 未授权或令牌无效
- `NOT_FOUND`: 资源不存在
- `INTERNAL_ERROR`: 服务器内部错误
- `VALIDATION_ERROR`: 数据验证失败
